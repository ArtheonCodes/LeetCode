## 🛠️ How I Approach Problems
1. **Understand:** Read constraints and edge cases.
2. **Brute Force:** Write the $O(n^2)$ logic first if stuck.
3. **Optimize:** Refine to $O(n \log n)$ or $O(n)$ using appropriate Data Structures.
4. **Document:** Explain the "Why" in the code comments.
